# 开发文档

本目录包含插件开发过程中的技术文档和问题修复记录,仅供开发者参考。

## 📚 文档清单

### 问题修复文档

- **CHINESE_ENCODING_FIX.md** - 中文字符编码问题修复记录
- **REPEAT_ORDER_FIX.md** - 重复下单问题修复记录
- **CODE_FIXES_SUMMARY.md** - 代码修复汇总报告

### 重构文档

- **REFACTORING_REPORT.md** - 代码重构详细报告
- **SERVICES_GUIDE.md** - 服务层使用指南

### 代码质量文档

- **CODE_QUALITY_AUDIT.md** - 代码质量审查报告

## 🎯 用途

这些文档记录了:
- 遇到的问题及根因分析
- 修复方案和实施过程
- 重构思路和架构设计
- 代码质量改进建议

## 📝 注意事项

- 这些文档**不应包含**在生产环境部署中
- 仅用于团队内部学习和参考
- 所有内容均已在Git历史中保存
