### 配置文件位置
- 主插件文件: `wpkj-fluentcart-alipay-payment.php`
- 配置类: `src/Config/AlipayConfig.php`
- 网关类: `src/Gateway/AlipayGateway.php`
- API 类: `src/API/AlipayAPI.php`
- 日志查看: FluentCart → Tools → Logs

### 常用命令
```bash
# 查看错误日志
tail -f /path/to/wordpress/wp-content/debug.log

# 清除缓存
wp cache flush

# 查看 FluentCart 日志
wp fluent-cart log list --payment_method=alipay

# 检查插件版本
wp plugin list | grep alipay
```
